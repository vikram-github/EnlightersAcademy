Courses Offered

Call Us Now 1-318-107-432

Please note this list is not exhaustive and only includes some of the most common curriculums. However if you require a tailored workshop please feel free to contact us.

  • Java: Understand the JVM architecture and insights of Java including OOPS, multi-threading, collections and concurrency
  • J2EE - Java Enterprise Edition: Understand Multi-Tiered architecture, Distributed Architecture, Web-Servers, App-Servers
  • Javascript: Understand Dynamic Programming Language, AJAX, DOM, Error Handling, Animations, Page printing
  • Android: Understand Mobile Operating System, Android Architecture, Building Native Mobile Applications, Deploying the mobile applications
  • ReactJS: Facebook's Framework to create interactive UI painless. Understand JSX, Component-Based, Building Web Applications using React JS
  • Angular JS: Google's Framework to create Rich and interactive UI painless. Understand Angular Expressions, Modules, Directives, Binding, Controllers and Building Web Applications using Angular JS
  • JQuery: Understand cross-platform javascript library, JQuery Plugins, Creating Rich UI interactions using JQuery
  • DOJO: IBM's Framework to create interactive and Rich UI. Understand DOJO Core, DOJOX, DOJO AMD, Building web applications using Dojo
  • Cloud Computing: Understand Cloud Computing Architecture, IaaS, PaaS, SaaS, public cloud, private cloud, Hybrid Cloud.
  • Big Data (Hadoop): Understand the current problems with Big Data, how to handle it using Hadoop, MapReduce, Hadoop Distributed File System, HBase, Hive, Apache Pig, Zoo-Keeper
  • NodeJS: Understand the Scripting on Server Side, Event Loop, Threading, Package Management, Unified API, NPM
  • MOM (Message Oriented Middleware): Understand messaging architecture, Interprocess communication, MOM providers (RabbitMQ, WebsphereMQ etc)
  • SQL: Understand messaging architecture, Interprocess communication, MOM providers (RabbitMQ, WebsphereMQ etc)
  • NoSQL: Understand Key-Value Store, Graph Store, Document Store, MultiValue and MultiModel DBs, Caching, Replication, Nesting, Samples using MongoDB or Apache Couche DB
  • Spring Framework: Understand Dependency Injection, Spring Core, Spring JDBC, Spring ORM, Spring MVC, Spring Containers, Spring Security
  • Hibernate Framework: Understand ORM with Java, Hibernate Mapping, Hibernate Query Language, Hibernate Entities and Components
  • Web Services: Understand SOAP (Simple Object Access Protocol), WSDL, REST, Comparison between REST and SOAP, Building web-services using REST
  • HTML5: Understand HTML5 Semantics, HTML5 Plugins, HTML5 Web-Storage, HTML5 Web-Workes, HTML5 Server-Sent Events
  • Networking: Understand Network Topology, Network Packet, Wired/Wireless Technologies, Network Nodes, Network Protocols, Network Security
  • Tools: MYSQL, Eclipse IDE, Tomcat Server, BracketsIO
